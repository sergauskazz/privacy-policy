О приложении

Данное приложение представляет собой Telegram-бота для автоматической публикации контента в социальных сетях (Instagram, ВКонтакте).

Какие данные мы собираем

  • Токены доступа к социальным сетям (Instagram, ВКонтакте)
  • Идентификаторы пользователей Telegram
  • Контент для публикации (тексты, изображения)
  • Настройки пользователей (часовой пояс, предпочтения)

Как мы используем данные

  • Исключительно для публикации контента в указанных социальных сетях
  • Для обеспечения функционала автопостинга
  • Данные не передаются третьим лицам
  • Не используются в коммерческих целях

Хранение данных

  • Данные хранятся локально в зашифрованном виде
  • Токены доступа обновляются автоматически
  • Пользователь может удалить свои данные в любое время

Права пользователей

  • Отзыв согласия на обработку данных
  • Удаление всех сохраненных данных
  • Получение информации о хранящихся данных

Контакты
